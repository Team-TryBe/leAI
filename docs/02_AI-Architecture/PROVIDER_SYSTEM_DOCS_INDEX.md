### Common Questions

**Q: Where do I create provider configs?**  
A: Admin Dashboard at `http://localhost:3000/admin/providers`

**Q: How are API keys stored?**  
A: Encrypted with Fernet (AES-256) - See security section in [`PROVIDER_MANAGEMENT_SYSTEM.md`](./PROVIDER_MANAGEMENT_SYSTEM.md)

**Q: What's the cost per provider?**  
A: See cost estimation in [`PROVIDER_INTEGRATION_GUIDE.md`](./PROVIDER_INTEGRATION_GUIDE.md#step-5-add-cost-estimation-helper)

**Q: How do I switch providers?**  
A: Edit config and change task routing checkboxes - [`PROVIDER_MANAGEMENT_QUICK_SETUP.md`](./PROVIDER_MANAGEMENT_QUICK_SETUP.md#common-tasks)

**Q: What if a provider fails?**  
A: System falls back to hardcoded model - See fallback pattern in [`EXAMPLE_PROVIDER_INTEGRATION.py`](./EXAMPLE_PROVIDER_INTEGRATION.py)
